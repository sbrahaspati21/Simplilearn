MySQL location is: C:\Program Files\MySQL\MySQL Server 5.1\bin
xampp location is: C:\xampp\mysql\bin

Steps:
1. open cmd 
2. cd C:\xampp\mysql
3. C:\xampp\mysql> path 
4. C:\xampp\mysql> set path=%PATH%;C:\xampp\mysql\bin;
5. C:\xampp\mysql> mysql -u root -p
